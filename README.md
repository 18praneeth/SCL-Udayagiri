**Installation**

Create a virtual env

In Windows
```bash
python -m venv myenv

myenv\Scripts\activate
```

In Ubuntu/MacOS
```bash
python -m virtualenv myenv

source myenv/bin/activate
```

Install all the requirements

```bash
pip install -r requirements.txt
```

Run the app.py
```bash
python app.py
```
